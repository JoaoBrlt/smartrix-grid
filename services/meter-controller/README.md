# Meter controller (Mock)

Allows services to send commands to electric meters.

## Requirements

- Install the dependencies:

```bash
npm install
```

## Development

- Run the service:

```bash
npm run start:dev
```

## Production

- Build the service:

```bash
npm run build
```

- Run the service:

```bash
npm run start:prod
```
