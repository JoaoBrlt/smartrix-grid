# Tests

## Preconditions

- The services are running.
- The database is empty.

## Requirements

- Install the dependencies:

```bash
npm install
```

- Copy the environment variables:

```bash
cp .env.example .env
```

## Run

- Run the tests:

```bash
npm run test
```
